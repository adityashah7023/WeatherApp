# The Weather App

Using React, Node, Express, [Open Weather Map](https://openweathermap.org) to build this weather app.

Live Demo: [https://festive-thompson-eef1e8.netlify.com/](https://festive-thompson-eef1e8.netlify.com/)

Install
-------
```bash
npm install
```

Run Express Server
------------------
```bash
npm run serve
```

Run React Client
----------------
```bash
npm run start
```
